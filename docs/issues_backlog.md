# Issues 초안

## 1. transcript 업로드 후 미팅 선택 UX 추가

라벨:
- enhancement

요약:
- 업로드한 파일 설명에 미팅 힌트가 없으면 `latest`에 바로 붙이지 않고 최근 미팅 후보를 먼저 보여준 뒤 사용자가 선택하게 한다.

완료 기준:
- `이 파일로 회의록 정리해줘` 입력 시 최근 미팅 후보 표시
- 사용자가 번호 또는 meeting_id로 답하면 그 미팅에 연결되어 처리

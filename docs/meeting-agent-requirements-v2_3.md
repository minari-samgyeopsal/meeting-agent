**0.1 에이전트 구조**

미팅 에이전트는 단일 오케스트레이터로 동작하며, 미팅 컨텍스트 내 작업은
직접 스킬로 처리하고 독립적인 대규모 작업은 전문 에이전트에 위임합니다.

  파이프라인**       MoU) → 수주 → Drop

  처리**

  **신규 카드 생성** 업체 카드 없으면 Contact/Meeting 리스트에 기본 생성

# Milestone 초안

## M1. Live MVP Stabilization

목표:
- 데모용 보정 없이 라이브 흐름이 안정적으로 동작하도록 정리

범위:
- transcript 업로드 후 미팅 선택 UX
- 회사명 / 참석자명 추론 안정화
- Slack 결과 포맷 개선
- Drive/Trello/Calendar 상태 정합성 정리

완료 기준:
- Slack에서 미팅 생성 → transcript 업로드 → 회의록/액션아이템/Trello까지 일관되게 동작

## M2. Knowledge & Briefing Quality

목표:
- 자사 정보와 고객 정보 기반의 브리핑/제안 품질 향상

범위:
- `company_knowledge.md` 체계화
- 회사/인물 문서 merge 품질 향상
- 브리핑 연결점 정확도 향상
- 제안서 초안 품질 개선

완료 기준:
- 브리핑과 제안서 초안이 회사/인물 문서와 자사 정보 기반으로 일관된 결과 제공

## M3. Conversational Slack UX

목표:
- 명령어형 봇에서 대화형 업무 비서에 가까운 UX로 개선

범위:
- intent router 고도화
- 애매한 요청 시 되묻기
- 업로드 파일과 미팅 자동 매칭
- 후속 액션 버튼/확인 UX 정리

완료 기준:
- 사용자가 짧은 자연어와 파일 업로드만으로 주요 업무 흐름 수행 가능
